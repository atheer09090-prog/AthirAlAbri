Final Year Project · Cybersecurity & ML

SmishGard — SMS Phishing Detection

SmishGard detects smishing (SMS phishing) attacks using a fine-tuned DistilBERT model trained on over 8,000 SMS messages. It achieved 99.5% accuracy, outperforming five other ML models including CNN-LSTM and XGBoost.

Every decision is explained using LIME — highlighting exactly which words triggered the detection — and cross-checks embedded links against the VirusTotal API in real time. Supports Arabic and English.
